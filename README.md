# Simple Task Management App

Aplikasi manajemen tugas sederhana sebagai bagian dari technical test
Program Internship PT Moonlay Technologies.

## Teknologi
- Backend: Python Flask
- Database: Firebase Firestore
- Frontend: HTML, CSS, JavaScript

## Fitur
- CRUD Task
- Status Task
- Activity Log
- REST API

## Endpoint API

### POST /tasks
Membuat task baru

### GET /tasks
Menampilkan seluruh task

### GET /tasks/<id>
Menampilkan task berdasarkan ID

### PUT /tasks/<id>
Memperbarui task

### DELETE /tasks/<id>
Menghapus task

## Cara Menjalankan Backend
1. Install dependencies
2. Konfigurasi Firebase
3. Jalankan `python app.py`

## Catatan
Proyek ini dibuat secara mandiri untuk keperluan seleksi magang.
